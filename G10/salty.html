The border-bottom shorthand property sets all the bottom border properties in one eclaration. The properties that can be set, are (in order): border-bottom-width, border-bottom-style, and border-bottom-color. It does not matter if one of the values above are missing, e.g. border-bottom:solid #ff0000; is allowed.
